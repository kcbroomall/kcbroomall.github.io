Hackopoly

Description

This is a true-to-the-original, full-featured reproduction of everyone's favorite Monopoly game.

Features

Features include email and Facebook authentication, sockets for real-time gameplay and chat, a lobby for messaging other players and entering game rooms, pausing games to be resumed later, and in-game cash and property trading.

Priorities

Prioritized solving 3 of Monopoly's biggest problems:

  • Slow Gameplay: Implemented a virtual banker, digital dice roll, automatic token movement, and timers to limit delays
  • Confusion Around Rules: Used the original Monopoly rule book to create and test for every possible user story
  • Unfinished Games: Created ability for users to store the state of a game for use at a later period

Technologies

- React - Redux - Node.js - Express - Socket.io - MySQL - MongoDB - Passport - Semantic UI -

  • Link: bit.ly/hackopoly
  • Github: Codebase
